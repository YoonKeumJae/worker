# 이미지 포맷 변환

## 목적

선택한 하나 이상의 이미지 파일을 같은 대상 포맷으로 일괄 변환한다.
새 복사본을 만들지 않고 기존 파일을 대상 확장자 파일로 교체한다.

## 지원 범위

- 입력 포맷: PNG, JPG, HEIC, HEIF, WebP
- 출력 포맷: PNG, JPG, HEIC, WebP
- 여러 파일 동시 선택
- 대상 확장자 충돌 사전 검사
- 확장자와 실제 이미지 포맷이 달라도 파일 내용 기준으로 읽기
- 이미 대상 포맷인 파일은 재인코딩하지 않고 건너뛰기
- 모든 선택 파일을 고유 임시 파일로 먼저 변환한 뒤 원본 교체
- 변환 실패 시 준비된 임시 파일 삭제와 원본 보존
- EXIF orientation 적용 후 저장
- JPG 출력 시 투명 픽셀은 흰색 배경으로 합성
- HEIC/HEIF 처리는 macOS `sips`를 사용
- WebP 처리는 Rust `image` WebP codec을 사용
- 애니메이션 WebP는 프레임 손실 방지를 위해 현재 변환하지 않음

## 동작

1. 사용자가 `이미지 변환` 도구를 연다.
2. 대상 포맷을 선택한다.
3. 하나 이상의 이미지 파일을 선택한다.
4. 원본 파일 교체를 확인한다.
5. `변환`을 실행한다.
6. 앱은 대상 확장자 파일명이 이미 존재하는지 먼저 확인한다.
7. 이미 대상 포맷인 파일은 건너뛰고, 나머지 파일은 대상 포맷으로 변환해 원본 경로를 대상 확장자 경로로 교체한다.

## 안전 규칙

- 선택 파일이 없으면 변환하지 않는다.
- 사용자가 원본 파일 교체를 확인하기 전에는 변환하지 않는다.
- 지원하지 않는 확장자는 변환하지 않는다.
- 대상 파일명이 중복되면 변환하지 않는다.
- 대상 경로에 기존 파일이 있으면 변환하지 않는다.
- 애니메이션 WebP는 변환하지 않는다.
- 변환 실패 시 임시 변환 파일은 삭제하고 원본 파일은 보존한다.
- 임시 파일은 충돌하지 않는 고유 파일명을 사용한다.
